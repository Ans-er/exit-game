# Mini Escape (Starter)

Ein minimales Webspiel (reines HTML/CSS/JS), ideal zum Testen mit **GitHub Pages**.

## Schnellstart
1. Repo auf GitHub erstellen, z. B. `escape-room` (Public).
2. Diese Dateien pushen (oder ZIP hochladen und entpacken).
3. **Settings → Pages**: *Deploy from a branch*, Branch **main**, Folder **/** (root).
4. Öffne `https://<dein-user>.github.io/escape-room/`.

## Dateien
- `index.html` – Einstieg & Layout
- `style.css` – einfache Styles
- `game.js` – sehr kleine Game-Logik

Erweitere `SCENES` in `game.js` um Räume, Items, Rätsel.
